# Legacy Practice

This folder contains older JavaScript and TypeScript practice files.

It is kept for reference while the cleaner learning path continues in `ReStart/`.

## Main Areas

- `array/`: array problems and patterns.
- `graph/`: graph representations and graph problems.
- `trees/`: binary tree and BST practice.
- `leetCode/`: older LeetCode solutions.
- `neetcode/`: older NeetCode-style solutions.
- `fundamentals/`: standalone JavaScript fundamentals moved from the repo root.
- `practice_in-dom/`: DOM and browser practice.
